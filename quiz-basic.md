# 前端开发的概念

## Q1

名词解释，前端开发的语境中，这些东西都是什么？

越精简越好，用一两句话你表现出的理解，例如：`Node.js 是 JS 的一个 runtime`  
（不了解的可以跳过）

请在下面的表格中作答：

| 名词           | 是什么               |
| :------------- | :------------------- |
| Node.js        | 是 JS 的一个 runtime |
| npm            | 是node package manager，node的包管理工具  |
| npx            | 是能执行依赖包里的二进制文件                |
| nvm            | 是 JS的  version manager              |
| nrm            | 是npm的注册表管理器                |
| yarn           | 是新的JS包管理工具，能并行执行任务                |
| React          | 是构建用户界面的JS库                |
| Vue            | 是构建用户界面的渐进式框架                |
| Flux           | 是构建客户端web应用的应用架构                |
| Mobx           | 是通过函数响应式编程使状态管理变得简单和可扩展的状态管理库        |
| MVVM           | 是一种软件架构设计模式               |
| TDD            | 是Test-Driven Development,是测试驱动开发                |
| Jest/Mocha     | 是常用的前端测试工具               |
| node_modules   | 是安装node后用来存放用包管理工具下载安装的包的文件夹               |
| package.json   | 是管理本地安装的npm包               |
| ECMAScript     | 是ECMA-262标准化的脚本程序设计语言               |
| Electron       | 是开发一个跨平台桌面应用的框架              |
| Babel          | 是工具链，主要用于将 es6+ 版本的代码转换为向后兼容的 JavaScript 语法                |
| Webpack        | 是一个module bundler(模块打包器)                |
| SSR            | 是Server-Side Rendering(服务器端渲染)               |
| BFF            | 是Back-end For Front-end的中间层               |
| Serverless     | 是对全部底层资源和运维工作的封装                |
| PWA            | 是渐进式网页应用（ProgressiveWebApp）               |
| V8             | 是Google的开源高性能JC引擎               |
| libuv          | 是事件驱动的I/O库，并且提供了跨平台（如windows, linux）的API                |
| Stack Overflow | 是一个与程序相关的IT技术问答网站               |
| Deno           | 是基于 V8构建的安全的TypeScript的Google运行时引擎               |
